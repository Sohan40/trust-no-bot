# Launch Plan

## Pre-launch goal

Get the first 20-50 users to complete at least one game and tell you whether they would play again.

Do not optimize for scale before the core loop is fun.

## Alpha launch

Audience:

- friends
- college groups
- builders who like AI experiments
- social deduction game fans

Alpha requirements:

- Classic Mode playable
- no login required
- mobile usable
- result page exists
- basic safety checks exist
- API cost limits exist

Questions to ask alpha users:

- Did you understand what to do?
- Did any character feel memorable?
- Did the vote feel uncertain?
- Would you play again tomorrow?
- Which part was boring?
- Which result would you share?

## Beta launch

Add before beta:

- Daily Challenge
- better result cards
- at least 8 character profiles
- basic analytics
- daily free limit
- better error handling

Beta channels:

- X/Twitter demos
- college WhatsApp groups
- AI builder communities
- short demo videos

## Launch content ideas

- "I played Mafia against AI. I trusted the wrong bot."
- "Can you catch the hidden AI opponent?"
- Daily challenge screenshots
- Result card comparisons

## Metrics

Track:

- landing to game start
- game start to completion
- replay rate
- share click rate
- average game duration
- user win rate
- AI cost per game

## Launch warning

Do not launch with too many modes. Too many unfinished modes make the product feel cheap.

Launch with one strong mode, then expand.
